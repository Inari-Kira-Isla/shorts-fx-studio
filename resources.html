本站自動化模組

Auto B-roll Module
broll_fetch.py — 文案自動配空鏡（已接入 Pexels/Pixabay API）
路徑：~/.openclaw/workspace/video-shorts/broll/broll_fetch.py

用法：
• 單關鍵字：python3 broll_fetch.py --query "sea urchin sushi" --count 3
• 整份文案逐句配片（中文自動翻英）：python3 broll_fetch.py --script script.txt --translate
• 抓完拼成一條 1080×1920 墊片：--stitch 4

原理：文案逐句 → Gemma 4 翻英文關鍵字 → Pexels/Pixabay API 搜直度片 → 下載 → ffmpeg 拼接。 正正係 YouTube 講解片「一句話配一段空鏡」嘅自動化做法。叫 Claude「/shorts-fx 幫我用 broll 配片」即可。
